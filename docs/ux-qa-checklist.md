# UX QA Checklist

## Bug Bash Session

- Duration: 2 hours.
- Test every page in sidebar order.
- Create one Trello card for every issue.
- Fix small issues during the session.
- Mark larger issues as P1 for next week.

## Accessibility Pass

- Check buttons and inputs have accessible names.
- Confirm Tab navigation reaches interactive controls in a logical order.
- Confirm focus state is visible.
- Check key text/background combinations against WCAG AA contrast.

## Mobile Test Matrix

- iPhone Safari, iOS 17+.
- Android Chrome, latest version.
- iPad or Android tablet.
- Capture screenshots for each major page and attach them to the QA notes.

## Print And QR Test

- Print QR on A4.
- Print QR on A5.
- Scan printed QR with a phone camera.
- Scan under bright, normal, and dim lighting.
- Confirm printed page contains only necessary content.
